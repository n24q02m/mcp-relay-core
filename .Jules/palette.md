## 2025-04-06 - Dynamic Helper Text & Required Indicators Accessibility
**Learning:** Adding a visual required indicator (`*`) is helpful for sighted users, but when the input already has a `required` attribute, screen readers will announce it twice. Adding `aria-hidden="true"` to the indicator prevents this. Additionally, when form fields have dynamic helper text or links generated by a schema, they must be given unique IDs and explicitly associated with the input field using `aria-describedby` so screen reader users hear the instructions when focusing the field.
**Action:** Always use `aria-hidden="true"` on purely visual required indicators if the input relies on the HTML5 `required` attribute. Use `aria-describedby` to programmatically associate dynamically generated hints, instructions, and help links with their respective input fields.
